## ✅ Completed Tasks

### 🤖 Task 1 — Rule-Based Chatbot
> **Folder:** [`Task1/`](./Task1/) &nbsp;|&nbsp; **Status:** ✅ Completed

Built a fully functional rule-based chatbot from scratch using **pure Python** — no external libraries, no AI models.

**How it works:**
- User input is lowercased and stripped
- Matched against a predefined `responses` dictionary
- Returns mapped response or falls back to `"I don't know!"`
- Exits cleanly on `bye`, `quit`, or `q`

**Key Highlights:**
- 100% deterministic — zero hallucination risk
- Fully offline — no API or internet dependency
- Demonstrates why rule-based systems outperform LLMs in accuracy-critical scenarios

```bash
# Run the chatbot
cd Task1
python task1.py
```
